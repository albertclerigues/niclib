niclib2.clamp_list(l, min_len=None, max_len=None)[source]¶

Clamps a given list so that its length is between min_len and max_len elements. To shorten a list elements are removed at regular intervals. To lengthen a list elements are repeated to reach desired length.

Parameters

  • l – the list

  • min_len – the minimum desired length

  • max_len – the maximum desired length

Returns

the clamped list
